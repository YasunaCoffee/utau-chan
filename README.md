# 🎤 うたうくん

**おんぷをならべて、うたわせてあそぼう！** — ブラウザだけで動く、ちいさなうた合成器（UTAU / ボーカロイド風おもちゃ）です。

ピアノロールに音符を置いて、ひらがなの歌詞を流し込むと、「うたうくん」が口をパクパクさせながら、その場で合成した声で歌ってくれます。インストール不要・音源データ不要。`index.html` をブラウザで開くだけで遊べます。

## ✨ できること

- 🎹 **ピアノロールで作曲** — クリックで音符を置く、ドラッグで移動、右端ドラッグで長さ変更、右クリックで削除
- 📝 **歌詞を流し込み** — ひらがな/カタカナの歌詞を書いて「流しこむ」ボタンを押すと、音符に1音ずつ割り当て。「ー」で伸ばし、「っ」でお休み、「にゃ」「きゃ」などの拗音もOK
- 🗣 **4人のこえ** — 🎀うたこ（ふつう）／🐣ちびすけ（たかい）／🤖ロボまる（きかい）／🎩おじさま（ひくい）
- 😊 **うたうくんの顔** — 歌に合わせて「あいうえお」の口の形でパクパク、まばたきもする
- 🎲 **おまかせ作曲** — ワンクリックでランダムなメロディ＋歌詞を自動生成してすぐ再生
- 📖 **プリセット曲** — きらきらのほし／ねこのうた／もふもふねこ（メロディはパブリックドメインの童謡、歌詞はオリジナル）
- ⬇ **WAV書き出し** — 作った歌をWAVファイルとして保存
- 🔗 **共有リンク** — 曲をURLに埋め込んでともだちに送れる（サーバー不要）
- 💾 **自動保存** — 編集内容はブラウザに自動保存

## 🚀 あそびかた

### いますぐ

```
git clone https://github.com/YasunaCoffee/utau.git
```

`index.html` をダブルクリックしてブラウザ（Chrome / Edge / Firefox / Safari）で開くだけ。

### GitHub Pages で公開する

リポジトリの **Settings → Pages → Source: Deploy from a branch → main / (root)** を選ぶと、
`https://<ユーザー名>.github.io/utau/` で誰でも遊べるようになります。共有リンク機能もそのまま使えます。

### 操作まとめ

| 操作 | 動き |
|---|---|
| マス目をクリック | 音符を置く |
| 音符をドラッグ | 移動（音の高さ・タイミング） |
| 音符の右端をドラッグ | 長さを変える |
| ダブルクリック | その音符の歌詞を変える |
| 右クリック / Alt+クリック | 消す |
| スペースキー | うたう / とめる |

## 🔬 しくみ

音声はすべて **Web Audio API のフォルマント合成**でリアルタイムに生成しています。録音された音源（音声ライブラリ）は一切使っていません。

1. 歌詞のかなをモーラに分解し、子音＋母音に変換（`きゃ` → k + 硬口蓋グライド + a）
2. 母音は、のこぎり波（声帯）を4本のバンドパスフィルタ（声道のフォルマント F1〜F4）に通して生成
3. 子音は、ノイズバースト（か行・た行）、摩擦ノイズ（さ行・は行）、鼻音ハム（な行・ま行）などをタイプ別に合成
4. ビブラート、ポルタメント（なめらかな音程移動）、ブレスノイズ、リバーブで「歌らしさ」を追加
5. 声の違いはフォルマント周波数のシフト＋ピッチシフト＋波形の変更で表現

音符データは16分音符グリッド上の `{start, len, midi, lyric}`。共有リンクはこれをJSON→Base64にしてURLハッシュに載せているだけなので、サーバーもデータベースも要りません。
