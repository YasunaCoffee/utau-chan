# 🎤 うたうくん

**AIが作曲・調教して、キャラクターが歌う** — テキストだけで操作できる、ちいさな歌声合成器(UTAU / ボーカロイド風)です。

人間用の作曲GUIはありません。譜面は「うたテキスト」というプレーンテキスト形式で、**AI(や人間)が1行ずつ書けます**。書いた譜面は:

- 🌐 `index.html` に貼れば、うたうくんが口をパクパクさせながら歌う(ブラウザだけで動作)
- 💻 `node sing.js 譜面.uta` でWAVファイルに(サーバー・依存ライブラリ不要)
- 🔗 URLに埋め込んでともだちに送れる(共有リンク)

声は録音データを一切使わず、フォルマント合成でその場で生成します。

## 🚀 すぐ聴く

```bash
git clone https://github.com/YasunaCoffee/utau.git
cd utau
node sing.js songs/kirakira.uta     # → songs/kirakira.wav ができる
```

またはブラウザで `index.html` を開いてサンプルボタン(⭐🐱🐑🌸🤖)を押すだけ。

GitHub Pages(Settings → Pages → main / root)で公開すれば、共有リンクごと誰でも遊べます。

## 📝 うたテキスト形式

```
# コメント
@tempo 104              テンポ(BPM)
@voice うたこ            うたこ / ちびすけ / ロボまる / おじさま
@style ふつう            べたうち / ふつう / こぶし / ささやき / げんき

ど4:き ど4:ら そ4:き そ4:ら | ら4:ほ ら4:し そ4:が:8
ふぁ4:よ ふぁ4:ぞ み4:ら み4:に | れ4:ひ れ4:か ど4:る:8~
```

**音符** は `高さ:歌詞[:長さ][調教記号]` を空白区切りで並べます。

| 要素 | 書き方 |
|---|---|
| 高さ | `ど れ み ふぁ そ ら し` + オクターブ(`ど4` = C4 = 真ん中のド)。`#`/`♭`、`C4` `F#4` 式もOK |
| 歌詞 | かな1音。`ー` のばす、`っ` 息つぎ、`にゃ` `きゃ` など拗音OK |
| 長さ | 16分音符いくつぶんか。省略=4(4分音符)。`ど4:ら:8` = 2分の長さのラ…ではなく8×16分=2拍 |
| 休符 | `・:4`(または `R:4`) |
| 小節線 | `\|` は読み飛ばされる飾り。見やすさのためにどうぞ |

### 🎛 調教

ボカロ文化の「調教」(べた打ちを人間らしく歌わせるピッチ・表情の作り込み)を、テキストで指定できます。

**スタイル(自動調教)** — `@style` で一括指定:

| スタイル | 効果 |
|---|---|
| `べたうち` | 無調教。しゃくり・ビブラート・ゆらぎ全部なし。ロボ感を愛でる用 |
| `ふつう` | ほどよいしゃくり・ビブラート・音程のゆらぎ |
| `こぶし` | 演歌。深いしゃくり、フレーズ末のフォール、長い音に自動でこぶし |
| `ささやき` | ブレス多め・小さめの声・浅いビブラート |
| `げんき` | 明るめの声・強めの抑揚・早めのビブラート |

**調教記号(手動調教)** — 音符の後ろに付けて1音ずつ指定:

| 記号 | 効果 |
|---|---|
| `<` | しゃくり(下から音程に入る) |
| `>` | フォール(語尾を落とす) |
| `~` | ビブラート強め |
| `_` | ビブラートなし |
| `*` | こぶし(音程の節回し) |
| `!` | アクセント(強く) |
| `?` | よわく |

例: `そ4:ら:8<~` = ソの音2拍、しゃくって入ってビブラート強め。

## 🌙 かたりモード(ポエトリーリーディング)

`@mode かたり` と書くと、音符なしで**ふつうの詩をそのまま**読んでくれます。

```
@mode かたり
@voice うたこ
@style ささやき

よるのまちを ひとりで あるいていた
ポケットのなかで ゆびさきだけが さむかった

ねえ、きょうのことは
ぜんぶ わすれてもいいけど。
```

- 抑揚は自動生成 — フレーズの入りは高く、終わりに向かって自然に下降し、モーラごとに音程が小さく揺れます
- `、` 小さな間 / `。` `!` 大きな間 / `?` は語尾が上がる / 空行はもっと大きな間 / スペースは息つぎ
- `@tempo` は1分あたりのモーラ数(省略時320。240でゆっくり、400で早口)
- 本文はひらがな/カタカナで(漢字は読み飛ばされます)

組み合わせのおすすめ: `@style ささやき` で囁き系ポエトリー、`@voice ロボまる` + `@style べたうち` で抑揚ゼロの機械よみ、`@voice おじさま` で渋い朗読。

## 🤖 AIにたのむ

このREADME(またはプレイヤー下部の「かきかた」)をAIに見せて、こんな風にたのめます:

> うたテキスト形式で「雨の日のカタツムリ」の歌を書いて。@voice ちびすけ、8小節、
> フレーズの入りにしゃくり、サビの伸ばす音にはビブラートを付けて調教もやって。

返ってきたテキストを `index.html` に貼るか、`.uta` ファイルに保存して `node sing.js` するだけです。

## 📁 ファイル構成

```
engine.js      うたエンジン(譜面パーサ + フォルマント合成DSP)。ブラウザ/Node両対応
sing.js        CLI: .uta → .wav
index.html     プレイヤー(貼って▶で再生、WAV保存、共有リンク、口パクキャラ)
songs/*.uta    サンプル譜面(調教スタイルのデモ含む)
```

## 🔬 しくみ

1. 歌詞のかなをモーラに分解し、子音+母音へ(`きゃ` → k + 硬口蓋わたり + a)
2. 母音はのこぎり波(声帯)を4本のバンドパスフィルタ(声道フォルマントF1〜F4)に通して生成
3. 子音はノイズバースト(か・た行)、摩擦ノイズ(さ・は行)、鼻音ハム(な・ま行)などをタイプ別に合成
4. 調教はすべて**ピッチ軌道の変形**として実装(しゃくり=下からの指数的アプローチ、こぶし=短い装飾振動、ビブラート=遅延付きLFO、ゆらぎ=低速ドリフト)
5. 仕上げにブレスノイズとシュローダー・リバーブ、ピーク正規化

すべて依存ゼロの素のJavaScript(`engine.js` 1ファイル)です。
