📝 歌詞（ひらがな / カタカナ）
あそびかた
・マス目をクリック → おんぷを置く
・ドラッグ → 移動 ／ 右はし → 長さ変更
・ダブルクリック → その音の歌詞を変える
・右クリック → けす
・スペースキー → うたう / とめる
・「ー」でのばし、「っ」でおやすみ
うたうくんの声は、Web Audio API のフォルマント合成でその場で作られています（音源データは使っていません）。
収録曲のメロディはパブリックドメインの童謡、歌詞はこのプロジェクトのオリジナルです。あそんだ曲は「共有リンク」でともだちに送れます。
